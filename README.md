# 📖 SmartInventoryCLI <a name="about-project"></a>

**SmartInventoryCLI** is a Ruby cli application that manage a commercial Iventory, where user can create new product items, update product, delete, make a general promotion or a single product promotion.

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

```
    ruby
```

### Setup

Clone this repository to your desired folder:

```bash
  git clone https://github.com/Graciano1997/SmartInventoryCLI.git
```
### Usage

Install this project with:

```bash
  cd SmartInventoryCLI
  ruby Main.rb
```
